## Perceptron Learning Algorithm(PLA)
    感知機是從生物神經細胞發想而來的。
    單個神經細胞結構大致為：樹突、突觸、細胞體、軸突，接收刺激後會根據是否超過某種閾值而有兩種狀態：活化和未活化，前者是刺激超過閾值，後者是沒超過，這項概念便可發展出感知機，解決二元分類問題。
